---
title: Grid
---

```markdown
<Grid cols=2>
    <!-- 2x2 grid of barcharts -->
    <BarChart ... />
    <BarChart ... />
    <BarChart ... />
    <BarChart ... />
</Grid>
```

## Options

| Name      | Description                                         | Possible Values          | Required | Default |
| --------- | --------------------------------------------------- | ------------------------ | -------- | ------- |
| `cols`    | Number of columns in the grid on a full size screen | 1,2,3,4,5,6              | no       | 2       |
| `gapSize` | Space between grid elements                         | "none", "sm", "md", "lg" | no       | "md"    |
