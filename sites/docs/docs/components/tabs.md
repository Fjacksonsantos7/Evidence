```markdown
    </Tab>
    <Tab label="Second Tab">
        Content of the Second Tab
    </Tab>
</Tabs>
```

## Options

### `Tab` Options

| Name  | Description                                                                                           | Required | Default |
| ----- | ----------------------------------------------------------------------------------------------------- | -------- | ------- |
| label | Label for the tab                                                                                     | Yes      | -       |
